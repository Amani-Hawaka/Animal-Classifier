# Animal-Classifier
AI tool that has been trained to classify animals. 3rd version of the classifier is able to determine whether the animal is a butterfly, snail, or fish. The accuracy can be improved by adding more images to the dataset.
# How to use:
1) Download the ZIP file
2) Unzip the files
3) Run aitraining
4) Run aitest
5) You have a functional AI animal classifier!
